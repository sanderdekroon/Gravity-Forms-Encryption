Encrypt all fields of the Gravity Forms entries.

== Description ==

Encrypt fields of Gravity Forms entries. Uses LibSodium as it's encryption library.

== Installation ==

= Minimum Requirements =

* PHP 7.3 or greater with LibSodium
* GravityForms 2.0 or greater

= Manual installation =

Download this plugin through Github and upload it to the webserver.
